# NopCommerce-testing-project
Manual testing project for NopCommerce e-commerce application, including test cases and functional testing.

## Objective
To validate the functionality of the application by creating and executing test cases.

## Testing Activities
- Test case design
- Test execution
- Validation testing
- Negative testing
- Boundary testing

## Modules Tested
- Login
- Registration
- Product Search
- Cart
- Checkout

## Tools Used
- MS Excel
- Browser
- ChatGPT (for test scenarios)

## Files Included
- Test cases
- Test scenarios
- Test results
